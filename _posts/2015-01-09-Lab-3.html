Read [this](http://www.pyzo.org/python_vs_matlab.html) ###Get a Github Account - Go to https://github.com/join to get your Github account. - Go to https://windows.github.com/ or https://mac.github.com/ to download the GitHub IDE. - You can get detailed information about the installation and usage by clicking the “help” button. ###Copy a repository - Download [this](https://github.com/materialsinnovation/project-pages) repository by clicking the “Download Zip” button at the bottom right of the page. - After you unzip it, you have a copy of the repository named “project-pages-gh-pages” locally. Open the GitHub IDE. If you are a mac user, click the “+” button and the “Add” button. If you are a windows user, click the “+” button and the “Create” button. Then choose the path where you just downloaded. ![Description](https://github.com/rcui9/Labs/blob/gh-pages/_posts/img/1.png?raw=true) ![Description](https://github.com/rcui9/Labs/blob/gh-pages/_posts/img/5.PNG?raw=true) - Click the “commit to master” at the bottom and click the “publish” button at the top right. ![Description](https://github.com/rcui9/Labs/blob/gh-pages/_posts/img/2.png?raw=true) - Create the “gh-pages” branch so as to build the github pages. ![Description](https://github.com/rcui9/Labs/blob/gh-pages/_posts/img/3.png?raw=true) - If you want to delete a repository, go to the page of the repository, click the “Settings” button at the right. In “Options”, click the “Delete this repository” at the bottom. ###Add a collaborator Now, you can find the repository on your github webpage https://github.com//. By clicking the “+” button at the top right, you can add collaborators to this repository. ###Add & Edit a file To do any changes to the repository, such as add or edit a file, you just need to modify the local repository. The GitHub will automatically detect the changes that you’ve made and you can synchronize the changes to the online repository by clicking the “commit to gh-pages” and “Sync” button using the GitHub. ![Description](https://github.com/rcui9/Labs/blob/gh-pages/_posts/img/4.png?raw=true) For example, - To add an image to the img folder, just add the new image to the local folder on your computer. Then commit and synchronize the changes using the GitHub IDE. - To edit a file, say config.yml, you can open it locally with a text editor, edit it. Then commit and synchronize the changes using the GitHub IDE. You can also edit a file by open it on your github webpages and make some changes. ![Description](https://github.com/rcui9/Labs/blob/gh-pages/_posts/img/6.png?raw=true) ###Setup project-pages To create your github pages, the only file that you need to modify is the config.yml. Make the changes accordding to the instructions in the comment. Then your Project Pages will be available at http(s)://.github.io/. It may take a few seconds for the webpage to appear. ###Make a post You can make a post by creating a .markdown file in the posts folder.
