--- layout: post title: "Lab 5" date: 2015-05-11 11:10:18 categories: Lab5 author: Yang Song markdownify: True tagline: Cloud Storage --- #Cloud Storage ###Why is github not used like dropbox for cloud storage? - Github is not as easy as Dropbox or similar services for that purpose. - Github could not save a large amount of data (over 1G). [Reference](https://help.github.com/articles/what-is-my-disk-quota/#file-and-repository-size-limitations) - The free accounts can only have public repositories in github, so you would also need to pay to keep your private documents on there. Dropbox and such do provide this for free. - The more public nature of Github makes us assume takedowns would be more frequent for illegal content sharing. - Github is not marketed towards this goal. Github offers all of the distributed revision control and source code management (SCM) functionality of Git as well as adding its own features. Most people would not think of using it for cloud storage. ###Cloud Storage Platforms (free) - Dropbox - One Drive - Google Drive - Flickr - Swift Stack (Gatech storm) ###Copy a repository - Download [this](https://github.com/materialsinnovation/project-pages) repository by clicking the “Download Zip” button at the bottom right of the page. - After you unzip it, you have a copy of the repository named “project-pages-gh-pages” locally. By clicking the “+” button at the top right, you can add collaborators to this repository. ###Add & Edit a file To do any changes to the repository, such as add or edit a file, you just need to modify the local repository. The GitHub will automatically detect the changes that you’ve made and you can synchronize the changes to the online repository by clicking the “commit to gh-pages” and “Sync” button using the GitHub. ![Description](https://github.com/rcui9/Labs/blob/gh-pages/_posts/img/4.png?raw=true) For example, - To add an image to the img folder, just add the new image to the local folder on your computer. Then commit and synchronize the changes using the GitHub IDE. - To edit a file, say config.yml, you can open it locally with a text editor, edit it. Then commit and synchronize the changes using the GitHub IDE. You can also edit a file by open it on your github webpages and make some changes. ![Description](https://github.com/rcui9/Labs/blob/gh-pages/_posts/img/6.png?raw=true) ###Setup project-pages To create your github pages, the only file that you need to modify is the config.yml.
